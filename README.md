# Fast-Pong
My version of the pong game 

![Fast-Pong](https://github.com/LuisOctavioGSeror/Fast-Pong/blob/main/Fast-Pong.gif)


## Requirements 

* Updated JDK

## Features 

* initial menu
* change dificult (even in the midle of the game)
* pause with menu (type esc to pause)
* nice songs effects ;)
